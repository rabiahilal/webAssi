Salifni: Item Exchange Database

Salifni is a database-backed application designed to facilitate item exchange and rental among users. The system ensures a robust and efficient structure for listing, searching, and managing items available for exchange or sale.

  • Technologies Used: SQL, Database Management, Web Integration
  • Features:
    • Item Listing and Search
    • Exchange and Rental Management
    • Database-Driven Operations
